# 3. 组件团队

## 3.1 Scrum开发团队

- 有5-9人组成
- 在项目期间专职投入，以跨职能的方式在一起工作
- 每个Sprint结束提交可工作的软件

![](imgs/3.1.1.png)

## 3.2 T型人才

- 按只是结构区分出来的人才类型
    - “——”表示有广播的只是面，“|”表示知识的深度
    - 即有较深的专业知识，又有广播的知识面
    - 一专多能
- Scrum Team需由T型人才组成
    - 木桶原理

    ![](imgs/3.1.2.png)
